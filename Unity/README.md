# UniPlay - Unity 게임 프로젝트

이 폴더는 UniPlay 플랫폼의 **Unity 기반 미니게임 프로젝트 소스**를 포함합니다.  

---

## 🎮 미니게임 구성

- 러닝 게임
- 슈팅 게임
- 수박 게임

각 게임은 Unity 엔진으로 제작되었으며, 플레이 결과(점수, 코인)는 웹 서버 및 DB와 연동됩니다.

---

## 🕹 주요 기능

- 캐릭터 이동 및 조작
- 게임별 점수 기록
- 코인 획득 시스템
- 아이템 적용 시스템
- PC / 모바일 빌드 지원 (소스 통합)

  ---

## 📁 프로젝트 구조

Unity/   
├─ Assets           : 게임 리소스 및 스크립트  
├─ Packages         : Unity 패키지 정보  
└─ ProjectSettings  : 해상도, 입력, 빌드 설정  

---

## 🛠 개발 환경

- OS : Windows 10
- Game Engine : Unity 2022 LTS
- IDE : Visual Studio 2022
- Language : C#
