# UniPlay – Database (PracticeDB)

이 폴더는 UniPlay 프로젝트의 MSSQL 데이터베이스 스키마와 리소스를 포함합니다.  
Unity 게임 클라이언트와 ASP.NET 웹 서버가 공통으로 사용하는 데이터의 기준이 되는 DB입니다.

---

## 🗄 데이터베이스 개요

- DBMS: Microsoft SQL Server (MSSQL)  
- 데이터베이스명: PracticeDB  
- 역할 및 용도:
  - 회원 정보 관리  
  - 게임, 점수, 랭킹 데이터 저장  
  - 아이템 / 인벤토리 관리  
  - 공지사항, 커뮤니티, 댓글, 좋아요 저장  

---

## 📁 폴더 구조

DB/  
├─ Tables/ — 테이블 생성 SQL 스크립트들  
├─ ERD/ — 전체 ERD 다이어그램 이미지  
├─ Backup/ — (선택) 전체 DB 백업 파일  
└─ README.md — 이 파일
