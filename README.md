# Project-Uniplay
Unity 기반 미니게임 + ASP.NET 기반 웹 사이트 플랫폼 UniPlay 프로젝트

---

## 프로젝트 소개
UniPlay는 Unity 기반 미니게임과 ASP.NET 기반 웹 사이트를 하나의 계정 시스템으로 연동한 게임 플랫폼 프로젝트입니다.  
웹에서는 회원 관리, 랭킹, 상점, 공지사항, 커뮤니티 게시판 기능을 제공하고,   
Unity에서는 미니게임 실행, 코인 획득, 아이템 적용 기능이 연동됩니다.  

---

## 주요 기능
- Unity 미니게임 3종
- 회원 가입 및 로그인
- 웹 기반 상점 및 커뮤니티 게시판 시스템
- 코인 기반 아이템 구매
- 인벤토리 및 아이템 장착 시스템
- 게임별 랭킹 시스템

---

## 모바일 지원

- Unity 기반 Android 빌드 지원
- PC와 모바일 동일 계정 연동
- 모바일에서도 로그인, 게임 플레이, 코인 획득 가능
- 웹 상점에서 구매한 아이템 모바일 게임에도 동일 적용

---

## 기술 스택

- HTML / CSS

Project-UniPlay

## 개발 목적
Unity 게임과 웹 시스템, 데이터베이스를 하나의 플랫폼으로 연동하는 구조를 직접 설계하고 구현하는 것을 목표로 진행한 프로젝트입니다.
